XR Room-scale Interfaces

Envisionment of room-scale interface combining tangible, multi-touch, and VR/AR interaction with multiple displays in a variety of shapes and sizes. Users can choose how to interact with the room-scale interface according to their needs. 


Partially functional prototype of the Enodia room. A constellation of screens, tangibles, and head-mounted displays allowed interaction with the digital content.

 

Combining Tangibles and Virtual Reality

A number of interfaces developed combine virtual reality, and tangible knobs. In this example, a Microsoft Surface Dial is co-located with the virtual reality dial. The user can interact with the physical dial while immersed in virtual reality.  

A Microsoft Surface Dial was embedded into a 3D printed body and tracked in virtual reality to create a fishing experience.  A tutorial on combining virtual reality and 3D printed artifacts was presented at the 2018 IEEE-VR conference.

In another exploration combining tangibles and virtual reality, a Microsoft Surface Dial was attached to the bottom of a mug to function as a virtual reality controller. This provided users with a novel way to navigate virtual content.  

Other Functional Tangible Devices

Old smartphones (~5 years old) were combined with 3D printed elements to recreate control pads for the classic pong game. Embedded metallic shims and stylus tips were employed to couple the knobs to the capacitive screens.

A Voxel8 3D printer was used to embed “wires’’ and create capacitive musical instruments. a) EtherOcarina is a flute-like instrument. b) EtherScores can be used to learn musical notation. c) Presentation with the instruments

a) shows a clock displaying the job scheduling of a high-performance computing node. The clock uses LED lights and electronic paper to present information. b and c) interfaces combining discs and a tablet to allow interaction with content.

Active and Hybrid Tangible Knobs

Several variations on Active Challenge Coins have been created. Electronic paper and LED rings provide the user with visual information. Variations on wired, and wireless active coins have been built.

This active dial contains a bluetooth module for data communication, a capacitive button for input, and a compass for rotation and direction sensing. The 24-LED ring and the buzzer are capable of providing visual and auditory feedback.

Hybrid knobs may contain multiple NFC tags embedded in their bodies. Tags can uniquely identify the dials, contain links for external content, and beyond. Embedded NFC tags are highlighted in red and blue.

Passive Capacitive Knobs

Several editions of tangible knobs were created using laser cut technology. Detection of ID, position, and orientation on capacitive  commodity multi-touch surfaces (e.g. tablets) is achieved with constellations of three touch points.

3D printed knobs require less components and can be reproduced quickly. Conductive (black) thermoplastic PLA is used to couple the human touch with the underlying capacitive screen to allow interaction.

Knobs combining laser cut and 3D printed elements  open multiple opportunities for customization. These include engraving and embedding laser cut elements into the 3D printed parts. 

Sample Applications for Tangible Knobs

A video player application has been created. It allows combinations of tangible and multi-touch interaction while providing the user with novel ways to interact and present content. Some dials represent functions, others content.

Entrada, a computationally-mediated scientific poster platform substitutes paper for large (approx. 55-inch) displays, and allows for multi-touch and tangible interaction, on one or multiple tablets.

Multiple children-oriented sample applications were developed which combine tangible and multi-touch interaction. Figure a) shows knobs used as  stamps to create compositions and b) a matching blocks game.
